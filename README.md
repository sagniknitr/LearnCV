# image_processing-gui
A Qt based GUI developed for testing OpenCV functions faster and comparing results easily.


# Installation
Very Simple to Build

```
cd LearnCV
make
./LearnCV
```

Use ```make clean``` to remove build files

# System Requirements
OpenCV version 4 or higher

Qt5
